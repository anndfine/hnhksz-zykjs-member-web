## ✨ 项目特色

### 🏗️ 模块化设计
- **六大核心模块**：科技社简介、小组介绍、项目展示、活动动态、特色联动、招新口号
- **组件化架构**：每个模块独立开发，便于维护和扩展
- **响应式布局**：完美适配桌面端、平板端和移动端

### 🎯 七大专业小组
- **科创与智造组**：硬核技术 · 竞赛 · 校园改造
- **机器人与电子组**：机器人编程 · 电子电路
- **航模组**：航模设计与飞行
- **青芽·生趣营**：生态营造 · 劳动教育 · 审美浸润
- **民俗·文创工坊**：传统民俗 · 数字制造 · 创意手作
- **AI与智能应用组**：探索人工智能技术，开发校园智能应用
- **拾光融媒体组**：记录 · 传播 · 运营

### 📸 丰富的多媒体展示
- **照片墙功能**：展示成员活动精彩瞬间
- **项目成果展示**：25+个可公开项目，涵盖6个技术领域
- **动态内容管理**：支持数据驱动的项目展示

## 🚀 快速开始

### 环境要求
- Node.js ^20.19.0 || >=22.12.0
- npm 或 yarn

### 安装与运行

```bash
# 克隆项目
git clone https://github.com/anndfine/hnhksz-kjs.git
cd hnhksz-kjs

# 安装依赖
npm install

# 开发模式运行
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

### 项目结构
```
src/
├── components/          # 组件目录
│   ├── Layout/         # 布局组件
│   └── Sections/       # 页面区块组件
├── data/               # 数据文件
│   ├── projects.ts     # 项目数据
│   └── photos.ts       # 照片数据
├── views/              # 页面视图
└── styles/             # 样式文件
```
